# Lesson 16: INSERT Basics

## Overview
INSERT adds new rows.

### MySQL Example
```sql
INSERT INTO employees (first_name, last_name, department_id, salary)
VALUES ('Karen', 'Lopez', 2, 58000);
```

### SQLite Example
```sql
INSERT INTO employees (first_name, last_name, department_id, salary)
VALUES ('Karen', 'Lopez', 2, 58000);
```
