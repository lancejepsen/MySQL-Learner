# SQL Case Study: HR Workforce & Project Analytics (SQLite Edition)

##Scenario
You are a new data analyst working with the HR department. Your mission is to analyze workforce, salary, and project data using SQLite.

## Database Schema (Schema A)
- employees(id, first_name, last_name, department_id, salary, hire_date)
- departments(id, department_name)
- projects(id, project_name, budget)
- employee_projects(employee_id, project_id)
